Sobre

Olá! Meu nome é Matheus Fernandes, e eu fiz esse site para praticar algumas tecnologias.

Antes de começar o desenvolvimento, eu defini meus requisitos, e, embora novas ideias aparecessem durante a codificação, mantive-os até o fim.

Os requisitos eram os seguintes:

  • Desenvolver uma One Page Application, usando angular.js
  • O site deve gerar proceduralmente sabores de bolos personalizados
  • A interface de usuário deve conter animações e transições 100% em CSS
  • Tanto os scripts quanto as folhas de estilo devem estar estruturadas em diretórios e devem ser carregadas pelo navegador como um único arquivo minificado.
  • Usar as ferramentas disponíves no npm

Abaixo segue uma especificação das tecnologias utilizadas.

Ambiente de desenvolvimento

  • Cloud9 como IDE;
  • Git como ferramenta de versionamento;
  • Gulp como automatizador de tarefas.

Javascript

  • Angular.js como framework;
  • Códigos concatenados e minificados pelo Gulp;
  • Simulação de uma API que retorna JSON
    • Bolos
    • Sabores

CSS

  • Sass como preprocessador, através do Gulp;
  • Flexbox Grid como sistema de grade;
  • TypeScale como gerador de escala tipográfica;
  • Ícones obtidos no FlatIcon;

É claro que existem detalhes a melhorar, mas acredito que atingi o meu objetivo de atender aos requisitos pré-definidos.
